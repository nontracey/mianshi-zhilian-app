# 面试智练隐私政策

生效日期：2026-06-11

面试智练是一款本地优先的技术面试主动回忆学习工具。我们尽量减少数据收集，默认把学习记录、练习记录、AI 配置和个人备考资料保存在你的设备上。

## 我们如何处理你的数据

### 本地保存的数据

以下数据默认保存在本地设备：

- 学习进度、掌握度、练习记录、模拟面试记录
- 备考目标、JD 解析结果、项目深挖资料、回答版本库
- 应用设置、主题、语言、内容源、线路偏好
- 你配置的 AI 服务地址、模型名称和 API Key。原生平台优先使用系统安全存储保存 API Key；Web 端受浏览器限制保存在本地站点存储。
- 可选同步配置，例如 WebDAV、GitHub、Gitee 的同步目标信息

### 会发送到外部服务的数据

只有在你主动使用相关功能时，数据才会发送到外部服务：

- 使用 AI 评估、AI 改写、图片分析或云端语音识别时，当前题目、你的回答、附件或音频会发送到你自己配置的 AI 服务。
- 使用 WebDAV、GitHub、Gitee 或文件同步时，同步快照会写入你选择的同步目标。默认同步快照不包含完整练习回答、回答版本、session feedback 和 AI 配置元数据；备考计划和项目库默认同步，如包含公司、JD 等敏感信息可手动关闭。
- 使用账号、工单、更新检查或内容加载功能时，客户端会请求面试智练的官方服务。

## API Key 和敏感信息

- API Key 默认保存在本地设备；原生平台优先写入 Keychain、Keystore 或 DPAPI。
- API Key 不会进入文件导出和同步快照。
- 复制诊断日志时，常见 Key、Authorization、token/password/secret 字段以及本机用户目录路径会自动脱敏。正式版不会把普通 `debugPrint` 调试输出写入用户可见日志。
- 如果你在第三方同步目标中手动保存了敏感信息，请自行管理该目标的访问权限。

## 同步边界

面试智练的同步快照只同步学习相关数据和必要配置。同步快照会跳过 API Key、登录态、同步目标凭证、缓存和运行态数据；完整回答、回答版本和 AI 配置元数据默认不进入同步快照。头像和备考计划默认同步，用户可按需关闭。更多技术边界见 [数据存储与同步边界](./data-storage-and-sync.md)。

## 账号和工单

如果你使用账号、工单或密码重置功能，相关用户名、登录令牌、工单内容、联系方式和必要的安全记录会由面试智练官方 API 处理，用于登录、反馈处理、密码重置人工核验和安全控制。密码重置提交成功时，接口只返回工单 id、类型、状态和创建时间，不回显联系方式和说明正文。

## 统计数据

应用默认上报基础使用统计，例如访问和功能使用计数，用于判断功能稳定性和改进优先级。你可以在「个人中心 → 同步与备份 → 匿名使用统计」中随时关闭；关闭后立即停止上报，并清空本机待发送的统计缓冲。统计不应包含你的 API Key 或完整练习回答；官方 API 的统计聚合表不保存登录用户 id。登录后的设备绑定记录仅用于账号安全、设备识别和限制滥用。

## 数据导出与删除

你可以在应用内导出本地数据，也可以清空本地学习/练习数据。卸载应用、清除浏览器站点数据或清除应用数据通常也会删除本地保存的数据。

## 第三方服务

你自行配置的 AI 服务、WebDAV、GitHub、Gitee 等第三方服务会按其各自隐私政策处理数据。使用前请确认你信任这些服务，并理解其数据处理方式。

## 政策更新

如果数据处理方式发生明显变化，我们会更新本政策，并在 README、Web 页面或版本说明中提示。

## 联系方式

如有隐私或数据安全问题，请通过 GitHub Issues 或项目主页提供的联系方式反馈。
