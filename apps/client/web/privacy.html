面试智练隐私政策

生效日期：2026-06-06

面试智练是本地优先的技术面试主动回忆学习工具。学习记录、练习记录、AI 配置和个人备考资料默认保存在你的设备上。

本地保存的数据

学习进度、掌握度、练习记录、模拟面试记录、备考目标、JD 解析结果、项目资料、回答版本库、应用设置、AI 配置和同步配置默认保存在本地设备。

会发送到外部服务的数据

  • 使用 AI 评估、AI 改写、图片分析或云端语音识别时，当前题目、你的回答、附件或音频会发送到你自己配置的 AI 服务。
  • 使用 WebDAV、GitHub、Gitee 或文件同步时，同步快照会写入你选择的同步目标。
  • 使用账号、工单、更新检查或内容加载功能时，客户端会请求面试智练的官方服务。

API Key 和敏感信息

API Key 默认保存在本地设备，不会进入文件导出和同步快照。复制诊断日志时，常见 Key 和 Authorization 字段会自动脱敏。

同步边界

同步快照只同步学习相关数据和必要配置，会跳过 API Key、登录态、同步目标凭证、缓存和运行态数据。

账号、工单与统计

如果你使用账号、工单或密码重置功能，相关用户名、登录令牌、工单内容和必要安全记录会由面试智练官方 API 处理。应用可能上报基础使用统计，用于判断功能稳定性和改进优先级。

数据导出与删除

你可以在应用内导出本地数据，也可以清空本地学习/练习数据。卸载应用、清除浏览器站点数据或清除应用数据通常也会删除本地保存的数据。

第三方服务

你自行配置的 AI 服务、WebDAV、GitHub、Gitee 等第三方服务会按其各自隐私政策处理数据。使用前请确认你信任这些服务。

联系与更新

如果数据处理方式发生明显变化，我们会更新本政策。如有隐私或数据安全问题，请通过 GitHub Issues 或项目主页提供的联系方式反馈。
